# 软件同步

将本地 Mac 上安装的软件同步到远程电脑。

## 远程电脑信息

| 配置项 | 值 |
|--------|-----|
| IP 地址 | 10.18.109.171 |
| 用户名 | jacky |
| 登录命令 | `ssh jacky@10.18.109.171` |

## 同步方式

由于远程电脑可能没有 Homebrew 或网络受限，优先使用 **DMG 传输安装**：

1. 本地下载 DMG 安装包
2. SCP 传输到远程电脑
3. 远程挂载并安装

---

## 已同步软件

### Zed 编辑器

**版本**：v0.225.13

**本地安装**（已完成）：
```bash
brew install --cask zed
```

**远程同步**（已完成）：

```bash
# 1. 本地下载 DMG
curl -L -o /tmp/Zed-aarch64.dmg \
  "https://github.com/zed-industries/zed/releases/download/v0.225.13/Zed-aarch64.dmg"

# 2. 传输到远程电脑
scp /tmp/Zed-aarch64.dmg jacky@10.18.109.171:/tmp/

# 3. 远程挂载并安装
ssh jacky@10.18.109.171 '
mkdir -p ~/Applications
hdiutil attach /tmp/Zed-aarch64.dmg
cp -R /Volumes/Zed/Zed.app ~/Applications/
hdiutil detach /Volumes/Zed
'

# 4. 清理临时文件
rm /tmp/Zed-aarch64.dmg
ssh jacky@10.18.109.171 'rm /tmp/Zed-aarch64.dmg'
```

**状态**：✅ 本地和远程均已安装

**远程启动方式**：
```bash
ssh jacky@10.18.109.171 'open ~/Applications/Zed.app'
```
