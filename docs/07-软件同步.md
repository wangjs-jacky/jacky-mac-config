# 软件同步

将本地 Mac 上安装的软件同步到远程电脑。

## 远程电脑信息

| 配置项 | 值 |
|--------|-----|
| IP 地址 | 10.18.109.171 |
| 用户名 | jacky |
| 登录命令 | `ssh jacky@10.18.109.171` |

## 前置条件

远程电脑需要先安装 Homebrew（**需要手动交互输入密码**）：

### 步骤 1：SSH 登录远程电脑

```bash
ssh jacky@10.18.109.171
```

### 步骤 2：安装 Homebrew

```bash
# 安装 Homebrew（需要网络通畅，会提示输入密码）
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

> **注意**：如果网络不通，需要先配置代理或使用其他方式下载安装脚本。

### 步骤 3：配置 PATH（Apple Silicon Mac）

```bash
echo 'eval "$(/opt/homebrew/bin/brew shellenv)"' >> ~/.zprofile
eval "$(/opt/homebrew/bin/brew shellenv)"
```

### 步骤 4：验证安装

```bash
brew --version
```

## 同步方式

由于远程电脑可能无法直接访问外网，有两种方式安装软件：

1. **远程直接安装**（需要网络通畅）
2. **本地下载后传输**（网络不通时使用）

## 已同步软件

### Zed 编辑器

**本地安装**（已完成）：
```bash
brew install --cask zed
```

**远程同步步骤**：

```bash
# 确保远程已安装 Homebrew，然后执行
ssh jacky@10.18.109.171 'brew install --cask zed'
```

> **状态**：本地已安装，远程待安装（需要先安装 Homebrew）

**验证安装**：
```bash
ssh jacky@10.18.109.171 'zed --version'
```
